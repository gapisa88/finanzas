Primera configuración. Introduce los datos de tu proyecto Firebase.

Pasos para configurar Firebase:
1. console.firebase.google.com → Crear proyecto (gratis)
2. Activa Authentication → Método de acceso → Email/contraseña ✓
3. Activa Realtime Database → Europa → modo test
4. En Realtime Database → Reglas, pega esto y publica:
{"rules":{".read":"auth != null",".write":"auth != null"}}

5. ⚙️ Configuración del proyecto → Tus apps → Web → copia los datos
6. Comparte este archivo HTML con Blanca (ella verá esta misma pantalla la primera vez)
